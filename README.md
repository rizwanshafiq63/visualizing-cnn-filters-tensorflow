# Visualizing Filters of a CNN using TensorFlow

Welcome to **Visualizing Filters of a CNN using TensorFlow**.
This is a project-based Coursera course designed to be completed in **less than 1 hour**.

## Course Objectives

The primary learning objective of this course is to **visualize image features that maximally activate filters of a Convolutional Neural Network (CNN)**.

## Course Structure

This course is divided into 3 parts:

1. **Course Overview**
   Introductory reading material.
2. **Visualizing Filters of a CNN using TensorFlow**
   The hands-on project.
3. **Graded Quiz**
   The final assignment you need to pass to complete the course successfully.

## Project Structure

The hands-on project is divided into the following tasks:

- **Task 1:** Introduction
- **Task 2:** Downloading the Model
- **Task 3:** Get Submodels
- **Task 4:** Image Visualization

## About This Guided Project

In this short, 1-hour guided project, we use the popular **VGG16** CNN model to visualize filters from different layers.
Using **gradient ascent**, we generate images that maximally activate specific filters across the network.

We use **TensorFlow** as the machine learning framework and **Google Colab** as the environment for running notebooks in the cloud (including free GPU access).
